La alineacion horizontal se hace con text-align, y sus propiedades son: left, center, right y justify

Este es un texto centrado con text-align:center;

Este es un texto alineado a la izquierda con text-align:left;

Este es un texto alineado a la derecha con text-align:right;

Este es un texto justificado con text-align:justify; pero OJO, para que esta propiedad funcione se necesita que el texto sea al menos de 2 lineas. La ultima linea siempre se alinea a la izquierda y a la anterior(es) se le aplica la justificacion. Si queremos que la última linea se justifique, se le aplica : text-align-last:justify;

La propiedad text-align-last especifica como alinear la última linea de un texto. 
    Esta propiedad puede tener uno de los siguientes valores:

        auto - Valor por defecto.La última linea se alinea a la izquierda.
        left - Se alnea a la izquierda (left).
        right - Se alinea a la derecha (right)
        center - Se alinea en el centro (center)
        justify - Se justifica como el resto de las lineas.
        start - La última linea se alinea a la izquierda de la linea. Es el valor por defecto.
        end - La ultima linea se alinea a la derecha ( end )

         start y end se usan principalmente al usar otros idiomas. Por ejemplo si se lee al reves nos interesa que la última linea se alinee a la derecha.
     
Vertical Alignment
La propiedad vertical-align fija el alineamiento vertical de un elemento.

La propiedad puede tener UNA de los siguientes valores:

baseline    - Default value. The element is aligned with the baseline of the parent
length/%    - Raises or lower an element by the specified length or percent
sub         - The element is aligned with the subscript baseline of the parent
super       - The element is aligned with the superscript baseline of the parent
top         - The element is aligned with the top of the tallest element on the line
text-top    - The element is aligned with the top of the parent element's font
middle      - The element is placed in the middle of the parent element
bottom      - The element is aligned with the lowest element on the line
text-bottom - The element is aligned with the bottom of the parent element's font
