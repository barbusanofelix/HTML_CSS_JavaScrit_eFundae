Responsive Web - Media Query

A responsive web design will automatically adjust for different screen sizes and viewports.

Key components in responsive web design are:

  1. Viewport <meta> tag
  2. Flexible layout (grid and flex)
  3. Media queries

1. Dentro de la etiqueta <head> hay que colocar la etiqueta:
<meta name="viewport" content="width=device-width, initial-scale=1.0"> 

            Desglose del código:
            name="viewport": Le indica al navegador que esta configuración es para el área de visualización.
    
            content="width=device-width": Esta es la clave. Le dice a la web: "Haz que el ancho de la página sea igual al ancho de la pantalla del dispositivo".
    
            initial-scale=1.0: Establece el nivel de zoom inicial (100%).
